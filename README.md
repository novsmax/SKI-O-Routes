# Цифровой ассистент для спортивного ориентирования на лыжах

Веб-приложение для анализа карт спортивного ориентирования, автоматического распознавания лыжных трасс и определения оптимальных маршрутов между контрольными пунктами.

## 📝 О проекте

Спортивное ориентирование — это вид спорта, где участники используют карты для навигации между контрольными пунктами. Этот проект создан для помощи спортсменам и тренерам в анализе выбора оптимальных путей при лыжном ориентировании.

### 🔑 Ключевые возможности

- Загрузка и хранение карт спортивного ориентирования
- Автоматическое распознавание лыжных трасс различного типа
- Построение графовой модели местности с учетом особенностей рельефа
- Нахождение оптимальных маршрутов между контрольными пунктами
- Визуализация результатов анализа

## 🔬 Технологии обработки изображений и поиска путей

### 🔍 Выделение лыжней из карты

Система использует многоэтапный процесс обработки изображений для выделения лыжных трасс:

1. **Цветовая фильтрация**
   - Выделение пикселей в заданном HSV-диапазоне (зеленые оттенки для лыжней)
   - Устранение больших однородных областей, не относящихся к трассам

2. **Обработка изображения**
   - Морфологические операции (дилатация, эрозия) для уточнения формы лыжней
   - Различение типов лыжней (скоростные сплошные и медленные пунктирные)
   - Достраивание пунктирных линий для создания связной сети трасс

3. **Скелетонизация**
   - Преобразование выделенных областей в одномерные центральные линии
   - Очистка от коротких ветвей и шумов

4. **Обнаружение перекрестков**
   - Поиск точек с тремя и более соседними пикселями
   - Кластеризация близких перекрестков для исключения дубликатов

```python
# Пример параметров HSV-фильтрации для выделения лыжней
HSV_PARAMS = {
    "h_min": 55,  # Минимальное значение оттенка (зеленый)
    "h_max": 70,  # Максимальное значение оттенка
    "s_min": 50,  # Минимальная насыщенность
    "s_max": 255, # Максимальная насыщенность
    "v_min": 50,  # Минимальная яркость
    "v_max": 215  # Максимальная яркость
}
```

### 📊 Построение графовой модели

На основе обработанного изображения система строит граф, где:

- **Вершины** — перекрестки лыжных трасс
- **Ребра** — сегменты лыжней между перекрестками
- **Веса ребер** — время прохождения участка, учитывая:
  - Длину сегмента
  - Тип лыжни (скоростная, обычная, медленная)
  - Рельеф местности (подъемы и спуски, если известны)

Процесс построения графа:
1. Идентификация вершин графа из координат перекрестков
2. Трассировка линий между перекрестками для определения ребер
3. Расчет весов ребер на основе длины лыжни
4. Удаление несвязанных компонентов для получения цельного графа

### 🛣️ Алгоритмы поиска оптимального маршрута

Для нахождения оптимального пути между контрольными пунктами используются:

- **Алгоритм Дейкстры** — базовый алгоритм для поиска кратчайшего пути
- Для упрощения визуального анализа, оптимальный маршрут накладывается на исходную карту

## 💻 Технический стек

### Backend
- **Python 3.8+**
- **FastAPI** — высокопроизводительный веб-фреймворк
- **SQLAlchemy** — ORM для работы с базой данных
- **OpenCV** — библиотека компьютерного зрения для обработки изображений
- **NetworkX** — библиотека для работы с графами и алгоритмами на графах
- **scikit-image** — дополнительные алгоритмы обработки изображений
- **NumPy** — работа с многомерными массивами и математические операции

### Frontend
- **HTML/CSS/JavaScript** — основа пользовательского интерфейса
- **Bootstrap 5** — фреймворк для стилизации и компонентов UI
- **Jinja2** — шаблонизатор для генерации HTML

### Хранение данных
- **SQLite** — легковесная база данных для хранения информации о картах
- **Pickle/JSON** — сериализация графовых моделей и данных о перекрестках

## 🚀 Установка и запуск

### Требования
- Python 3.8 или выше
- Pip (менеджер пакетов Python)
- Виртуальное окружение (рекомендуется)

### Шаги установки

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/novsmax/SKI-O-Routes.git
   cd ski-o-assistant
   ```

2. Создайте и активируйте виртуальное окружение:
   ```bash
   python -m venv venv
   source venv/bin/activate  # На Linux/Mac
   venv\Scripts\activate     # На Windows
   ```

3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

4. Запустите приложение:
   ```bash
   python run.py
   ```

5. Откройте в браузере:
   ```
   http://127.0.0.1:8000
   ```

## 🖥️ Использование

1. **Загрузка карты**
   - На главной странице заполните форму с информацией о карте
   - Загрузите изображение карты

2. **Анализ карты**
   - На странице просмотра карты нажмите "Анализировать карту"
   - Система автоматически выделит лыжни и создаст графовую модель

3. **Построение маршрута**
   - Нажмите "Построить маршрут" и выберите начальную и конечную точки
   - Система найдет оптимальный маршрут и отобразит его на карте

4. **Библиотека карт**
   - В разделе "Библиотека карт" доступны все загруженные карты
   - Используйте фильтры для поиска нужных карт

## 🧩 Архитектура приложения

Приложение имеет модульную архитектуру:

1. **Модуль ввода и предобработки** (`app/routers/maps.py`)
   - Загрузка и управление картами

2. **Модуль анализа лыжней** (`app/trail_analyzer.py`)
   - Выделение лыжней из изображения карты

3. **Модуль построения графа** (`app/graph_builder.py`)
   - Создание графовой модели лыжной сети

4. **Модуль расчета маршрутов** (`app/graph_builder.py`, функции `find_optimal_route`)
   - Поиск оптимальных путей между контрольными пунктами

5. **Сервисный слой** (`app/services/map_processor.py`)
   - Координация всех этапов обработки карт
